Политика конфиденциальности приложения "Baby Tracker"

Дата вступления в силу: 27.11.2025

1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки и защиты информации о физических лицах (далее – Пользователи), пользующихся мобильным приложением "Baby Tracker" (далее – Приложение).

Оператором персональных данных является Рытов Сергей Юрьевич (ИНН 701723942967, Россия, Томская область).

Использование Приложения означает безоговорочное согласие Пользователя с настоящей Политикой и указанными в ней условиями обработки его персональной информации.

2. Состав собираемых данных

2.1. В рамках использования Приложения Оператор может обрабатывать следующие данные:

  • Данные учетной записи (Yandex ID): имя, email, аватар (получаемые через Yandex Auth SDK).
  • Данные о ребенке: имя, дата рождения.
  • Специальные категории персональных данных: данные о физиологических процессах ребенка (сон, кормление, гигиена, здоровье, рост, вес), вносимые Пользователем.
  • Технические данные: IP-адрес, модель устройства, версия ОС, FCM-токен (для уведомлений).

3. Цели обработки данных

3.1. Оператор собирает и хранит только ту информацию, которая необходима для:

  • Предоставления функционала Приложения (ведение дневника, построение графиков).
  • Синхронизации данных между устройствами Пользователя через облачное хранилище (Yandex.Disk/Yandex Cloud).
  • Предоставления функции "Совместный доступ" (Family Sharing) другим пользователям по приглашению.
  • Отправки Push-уведомлений (напоминания о сне/кормлении).

4. Правовые основания обработки

Обработка данных осуществляется на основании ст. 24 Конституции РФ, ст. 6 и ст. 10 Федерального закона №152-ФЗ «О персональных данных», а также согласия Пользователя на обработку персональных данных (в том числе специальных категорий), выраженного путем проставления галочки при первом запуске Приложения.

5. Передача данных третьим лицам

5.1. Приложение использует сторонние сервисы для обеспечения работоспособности:

  • ООО «ЯНДЕКС» (Yandex Cloud, Yandex ID): для аутентификации и облачного хранения резервных копий данных.
  • Google LLC (Firebase): исключительно для доставки Push-уведомлений (FCM) и аналитики сбоев (Crashlytics).

5.2. При использовании функции "Пригласить партнера" Пользователь самостоятельно и добровольно предоставляет доступ к данным ребенка третьему лицу (Партнеру). Оператор не несет ответственности за действия Партнера с этими данными.

6. Хранение и безопасность

Данные хранятся в зашифрованном виде в облачной инфраструктуре (Яндекс.Облако) на территории РФ. Локальные данные на устройстве могут быть удалены пользователем в любой момент через настройки Приложения ("Очистить локальные данные").

7. Права пользователя

Пользователь имеет право в любой момент отозвать согласие на обработку данных, удалив свой аккаунт через настройки Приложения (кнопка "Удалить аккаунт и все данные").
